# Poker 网页客户端

该目录提供一个简易的网页大厅和前端客户端，负责与已有的 `poker.exe <game_mode>` 后端控制台程序通信。

## 运行方式

```bash
npm install
npm start
```

默认会在 `http://localhost:3000` 提供网页。创建房间时会在当前目录尝试启动 `./poker.exe <game_mode>`，如果未找到可执行文件会在页面提示但不会影响网页运行。

## 功能概览

- 登录 / 注册：用户密码以 SHA-256 哈希保存在 `./data/user_data.json` 中。
- 大厅：选择拖拉机或德州扑克模式，查看房间列表、创建或加入房间。
- 房间：玩家可准备，房主在人数满足 `player_number` 约束后开始游戏，人数过多时可选择进入对局的玩家，其余玩家会被送回主页。
- 对局：房主启动后向控制台请求发牌 (`init_hand`)，根据发牌顺序为玩家分配手牌。支持出牌、校验错误、请求一轮得分 (`point`)，并在收到 `end` 信号后重置状态。房主可随时结束牌局并发送 `exit`。

前端使用 WebSocket 推送房间状态、出牌和得分事件，界面按逆时针方式布局玩家头像、得分和弃牌堆。
